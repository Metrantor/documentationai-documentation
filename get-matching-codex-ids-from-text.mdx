## `public.get_matching_codex_ids_from_text(p_series_id text, p_text text)`

### Zweck

Ermittelt aus einem freien Text die relevanten **normalen Codex-Eintrags-IDs** einer Serie.

### Was die Funktion fachlich macht

- durchsucht Codex-Namen

- durchsucht aktive Aliasse

- prüft Wortgrenzen

- verhindert überlappende Matches

- berücksichtigt Exclusions

- zieht **direkte** Relationen (`1-Hop`) nach

### Aufrufparameter

| Parameter     | Typ    | Bedeutung                              |
| ------------- | ------ | -------------------------------------- |
| `p_series_id` | `text` | Serien-ID, in deren Codex gesucht wird |
| `p_text`      | `text` | Freitext, der analysiert wird          |

### Rückgabewert

| Rückgabe | Typ      | Bedeutung                           |
| -------- | -------- | ----------------------------------- |
| Ergebnis | `uuid[]` | Liste relevanter Codex-Eintrags-IDs |

### Beispiel

```typescript
select public.get_matching_codex_ids_from_text(
  '95ea7620-8d91-4fb8-9fb3-82fe78125141',
  'Der Reaktor der Kalega wurde beschädigt. Captain Sarvin reagierte sofort.'
);
```

### Typisches Ergebnis

```typescript
{
  "11111111-1111-1111-1111-111111111111",
  "22222222-2222-2222-2222-222222222222"
}
```
