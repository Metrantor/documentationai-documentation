## `public.build_full_codex_context_json(p_beat_id uuid, p_text text)`

# `public.build_full_codex_context_json(...)`

## Zweck

Diese Funktion baut den **gesamten Prompt-Kontext** für einen Beat in einem einzigen Aufruf.

Sie liefert:

- Standard-Codex

- Super-Codex

- Chapter-Codex

- World-Codex

sowohl als:

- ID-Mengen

- JSONB-Payloads

- gerenderte Prompt-Blöcke

---

## Signatur

```typescript
public.build_full_codex_context_json(
  p_beat_id uuid,
  p_text text,
  p_max_relation_depth integer default 2
) returns jsonb
```

---

## Aufrufparameter

| Parameter              | Typ                    | Bedeutung                                                             |
| ---------------------- | ---------------------- | --------------------------------------------------------------------- |
| `p_beat_id`            | `uuid`                 | Beat, aus dem Kapitel und Serie automatisch abgeleitet werden         |
| `p_text`               | `text`                 | Freitext für das Textmatching                                         |
| `p_max_relation_depth` | `integer`, Default `2` | Maximale Rekursionstiefe für die Relationsauflösung im Basis-Matching |

---

## Wie die Funktion arbeitet

### Schritt 1: Beat-Kontext laden

Aus `p_beat_id` werden automatisch geladen:

- &#x20;Szene&#x20;

- &#x20;Kapitel&#x20;

- &#x20;Buch&#x20;

- &#x20;Serie&#x20;

### Schritt 2: Standard-Basis bestimmen

Über:

```typescript
get_matching_codex_ids_from_text(series_id, text, p_max_relation_depth)
```

### Schritt 3: Super-Codex ableiten

Über:

```typescript
get_super_codex_ids(base_ids)
```

### Schritt 4: Super-Codex filtern

Super-Codex-Ziele werden nochmals gegen Exclusions geprüft.

### Schritt 5: Standard-Codex bereinigen

Einträge, die im Super-Codex vorkommen, werden aus dem Standard-Codex entfernt.

### Schritt 6: Chapter-Codex bestimmen

Es werden alle am Kapitel verknüpften Codex-Einträge geladen:

- &#x20;nur aktive Einträge&#x20;

- &#x20;Exclusions gegen den Text&#x20;

- &#x20;Dublettenbereinigung gegen Super-Codex&#x20;

### Schritt 7: World-Codex bestimmen

Es werden alle `always_include = true` Codex-Einträge der Serie geladen:

- &#x20;nur aktive Einträge&#x20;

- &#x20;Exclusions gegen den Text&#x20;

- &#x20;Dublettenbereinigung gegen Super-Codex&#x20;

### Schritt 8: Für alle vier Mengen Payloads bauen

Über:

```typescript
get_prompt_codex_payload_for_beat(...)
```

### Schritt 9: Für alle vier Mengen Rendering bauen

Über:

```typescript
render_codex_payload(...)
```

# SQL

```typescript
create or replace function public.build_full_codex_context_json(
  p_beat_id uuid,
  p_text text,
  p_max_relation_depth integer default 2
)
returns jsonb
language plpgsql
stable
as $function$
declare
  v_series_text text;
  v_chapter_id uuid;
  v_result jsonb;
begin
  if p_max_relation_depth is null or p_max_relation_depth < 0 then
    p_max_relation_depth := 2;
  end if;

  select
    n.series_id::text,
    ch.id
  into
    v_series_text,
    v_chapter_id
  from public.beats b
  join public.scenes s
    on s.id = b.scene_id
  join public.chapters ch
    on ch.id = s.chapter_id
  join public.novels n
    on n.id = ch.novel_id
  where b.id = p_beat_id;

  if not found then
    raise exception 'Beat % wurde nicht gefunden', p_beat_id;
  end if;

  with params as (
    select
      coalesce(p_text, '') as raw_text,
      lower(coalesce(p_text, '')) as text_lower
  ),
  standard_base as (
    select coalesce(
      public.get_matching_codex_ids_from_text(
        v_series_text,
        coalesce(p_text, ''),
        p_max_relation_depth
      ),
      '{}'::uuid[]
    ) as ids
  ),
  super_raw as (
    select coalesce(
      public.get_super_codex_ids((select ids from standard_base)),
      '{}'::uuid[]
    ) as ids
  ),
  super_filtered as (
    with super_input as (
      select distinct unnest((select ids from super_raw)) as codex_id
    )
    select coalesce(
      array(
        select si.codex_id
        from super_input si
        join public.codex_entries ce
          on ce.id = si.codex_id
        cross join params p
        where ce.is_active = true
          and not exists (
            select 1
            from public.codex_exclusions ex
            where ex.entry_id = si.codex_id
              and p.text_lower like '%' || coalesce(ex.normalized_excluded_phrase, lower(ex.excluded_phrase)) || '%'
          )
        order by si.codex_id
      ),
      '{}'::uuid[]
    ) as ids
  ),
  standard_final as (
    select coalesce(
      public.remove_super_codex_duplicates(
        (select ids from standard_base),
        (select ids from super_filtered)
      ),
      '{}'::uuid[]
    ) as ids
  ),
  chapter_base as (
    select coalesce(
      array(
        select distinct ce.id
        from public.chapter_codex_entries cce
        join public.codex_entries ce
          on ce.id = cce.codex_entry_id
        cross join params p
        where cce.chapter_id = v_chapter_id
          and ce.is_active = true
          and not exists (
            select 1
            from public.codex_exclusions ex
            where ex.entry_id = ce.id
              and p.text_lower like '%' || coalesce(ex.normalized_excluded_phrase, lower(ex.excluded_phrase)) || '%'
          )
        order by ce.id
      ),
      '{}'::uuid[]
    ) as ids
  ),
  chapter_final as (
    select coalesce(
      public.remove_super_codex_duplicates(
        (select ids from chapter_base),
        (select ids from super_filtered)
      ),
      '{}'::uuid[]
    ) as ids
  ),
  world_base as (
    select coalesce(
      array(
        select distinct ce.id
        from public.codex_entries ce
        cross join params p
        where ce.series_id = v_series_text
          and ce.is_active = true
          and ce.always_include = true
          and not exists (
            select 1
            from public.codex_exclusions ex
            where ex.entry_id = ce.id
              and p.text_lower like '%' || coalesce(ex.normalized_excluded_phrase, lower(ex.excluded_phrase)) || '%'
          )
        order by ce.id
      ),
      '{}'::uuid[]
    ) as ids
  ),
  world_final as (
    select coalesce(
      public.remove_super_codex_duplicates(
        (select ids from world_base),
        (select ids from super_filtered)
      ),
      '{}'::uuid[]
    ) as ids
  ),
  standard_payload as (
    select coalesce(
      public.get_prompt_codex_payload_for_beat(
        (select ids from standard_final),
        p_beat_id
      ),
      '[]'::jsonb
    ) as payload
  ),
  super_payload as (
    select coalesce(
      public.get_prompt_codex_payload_for_beat(
        (select ids from super_filtered),
        p_beat_id
      ),
      '[]'::jsonb
    ) as payload
  ),
  chapter_payload as (
    select coalesce(
      public.get_prompt_codex_payload_for_beat(
        (select ids from chapter_final),
        p_beat_id
      ),
      '[]'::jsonb
    ) as payload
  ),
  world_payload as (
    select coalesce(
      public.get_prompt_codex_payload_for_beat(
        (select ids from world_final),
        p_beat_id
      ),
      '[]'::jsonb
    ) as payload
  )
  select jsonb_build_object(
    'input', jsonb_build_object(
      'series_id', v_series_text,
      'beat_id', p_beat_id,
      'chapter_id', v_chapter_id,
      'text', coalesce(p_text, ''),
      'max_relation_depth', p_max_relation_depth
    ),
    'ids', jsonb_build_object(
      'standard_codex_ids', to_jsonb((select ids from standard_final)),
      'super_codex_ids', to_jsonb((select ids from super_filtered)),
      'chapter_codex_ids', to_jsonb((select ids from chapter_final)),
      'world_codex_ids', to_jsonb((select ids from world_final))
    ),
    'payloads', jsonb_build_object(
      'standard_codex', (select payload from standard_payload),
      'super_codex', (select payload from super_payload),
      'chapter_codex', (select payload from chapter_payload),
      'world_codex', (select payload from world_payload)
    ),
    'rendered', jsonb_build_object(
      'standard_codex', public.render_codex_payload((select payload from standard_payload), 'codex'),
      'super_codex', public.render_codex_payload((select payload from super_payload), 'super codex'),
      'chapter_codex', public.render_codex_payload((select payload from chapter_payload), 'chapter codex'),
      'world_codex', public.render_codex_payload((select payload from world_payload), 'world codex')
    )
  )
  into v_result;

  return v_result;
end;
$function$;
```
