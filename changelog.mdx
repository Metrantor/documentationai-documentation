## Recent Updates

AuraWrite evolves quickly. You receive new features, performance improvements, and bug fixes with each release. Review the highlights below to understand changes and upgrade smoothly.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added real-time collaboration for multi-user editing sessions.
- Introduced AI-powered content suggestions with customizable prompts.
- New export options including PDF and Markdown with table of contents.

## Improvements

- Enhanced editor performance by `>30%` for large documents.
- Better mobile responsiveness in the dashboard.

## Bug Fixes

- Fixed sync issues when switching between devices.
- Resolved formatting loss in bullet lists during copy-paste.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Integrated grammar and style checking powered by advanced NLP models.
- Customizable themes and dark mode support.
- Keyboard shortcuts for power users (e.g., `Ctrl+K` for quick search).

## Improvements

- Streamlined onboarding with interactive tutorials.
- Reduced API latency for content generation requests.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Initial Release

- Core writing assistant with AI text generation.
- Document management and version history.
- Basic integrations with `https://api.example.com` for custom workflows.

## Breaking Changes

- Updated authentication to use JWT tokens instead of API keys.

</Update>

## Upgrading AuraWrite

Follow these steps to update to the latest version. Always back up your data first.

<Steps>
  <Step title="Backup Data" icon="database">

    Export your documents to avoid data loss.

    <CodeGroup tabs="CLI,Web">
      ```bash
      aura-write export --all --format=json
      ```
      ```bash
      # Visit dashboard.example.com/export
      ```
    </CodeGroup>

  </Step>
  <Step title="Update Package" icon="download">

    Run the update command based on your package manager.

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm update aura-write@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```bash
        yarn upgrade aura-write@latest
        ```
      </Tab>
    </Tabs>

  </Step>
  <Step title="Verify Installation" icon="check-circle">

    Test the new features.

    ```bash
    aura-write --version
    aura-write doctor
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Check the [migration guide](/introduction#upgrading) for version-specific instructions if you skip releases.
</Callout>
