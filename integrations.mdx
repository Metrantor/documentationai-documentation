## Overview

AuraWrite supports seamless integrations with popular third-party tools, webhooks for automation, and a robust API for custom solutions. You can sync documents, automate workflows, and extend functionality without leaving your environment. Start with pre-built connections or build your own using webhooks and API endpoints.

<Callout kind="info">
  All integrations require an active AuraWrite account. Generate your API key from the dashboard at `https://dashboard.example.com/settings/api`.
</Callout>

## Webhook Setup

Set up webhooks to receive events like document creation or updates. This enables automation with your preferred services.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to `https://dashboard.example.com/settings/webhooks` and click "New Webhook".
  </Step>
  <Step title="Configure Events" icon="settings">
    Select events such as `document.created` or `document.updated`.
  </Step>
  <Step title="Add Endpoint" icon="link">
    Enter your webhook URL, e.g., `https://your-webhook-url.com/aurawebhook`.
  </Step>
  <Step title="Verify Setup" icon="check-circle">
    Test the webhook. AuraWrite sends a `POST` request with event data.
  </Step>
</Steps>

Here is a sample webhook payload:

```json
{
  "event": "document.updated",
  "document": {
    "id": "doc_123abc",
    "title": "Project Roadmap",
    "updated_at": "2024-10-15T10:30:00Z"
  }
}
```

## API Access for Custom Integrations

Build custom solutions using AuraWrite's REST API at `https://api.example.com/v1`.

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/documents', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'New Document',
      content: 'Hello from integration!'
    })
  });
  const data = await response.json();
  console.log(data);
  ```
  ```python
  import requests

  headers = {
      'Authorization': f'Bearer {YOUR_API_KEY}',
      'Content-Type': 'application/json'
  }
  data = {
      'title': 'New Document',
      'content': 'Hello from integration!'
  }
  response = requests.post('https://api.example.com/v1/documents', json=data, headers=headers)
  print(response.json())
  ```
  ```bash
  curl -X POST https://api.example.com/v1/documents \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "title": "New Document",
      "content": "Hello from integration!"
    }'
  ```
</CodeGroup>

<ParamField path="title" param-type="string" required="true">
  Document title.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

## Cloud Storage Sync

Sync your AuraWrite documents with cloud providers using built-in connectors.

<Tabs>
  <Tab title="Google Drive" icon="cloud">
    Connect your Google account via OAuth.

    <Steps>
      <Step title="Authorize">
        Go to `https://dashboard.example.com/integrations/drive` and sign in.
      </Step>
      <Step title="Select Folder">
        Choose a Drive folder for two-way sync.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Dropbox" icon="package">
    Use API token for secure access.
