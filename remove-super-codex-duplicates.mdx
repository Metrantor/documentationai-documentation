## `public.remove_super_codex_duplicates(p_base_codex_ids uuid[], p_super_codex_ids uuid[])`

### Zweck

Entfernt aus den normalen Codex-IDs alle IDs, die zugleich im Super-Codex vorkommen.

### Was die Funktion fachlich macht

- dedupliziert beide Mengen

- bildet:

  - `base_ids - super_ids`

### Aufrufparameter

| Parameter           | Typ      | Bedeutung         |
| ------------------- | -------- | ----------------- |
| `p_base_codex_ids`  | `uuid[]` | Normale Codex-IDs |
| `p_super_codex_ids` | `uuid[]` | Super-Codex-IDs   |

### Rückgabewert

| Rückgabe | Typ      | Bedeutung                     |
| -------- | -------- | ----------------------------- |
| Ergebnis | `uuid[]` | Bereinigte Standard-Codex-IDs |

### Beispiel

```typescript
select public.remove_super_codex_duplicates(
  array[
    '11111111-1111-1111-1111-111111111111'::uuid,
    '22222222-2222-2222-2222-222222222222'::uuid
  ],
  array[
    '22222222-2222-2222-2222-222222222222'::uuid
  ]
);
```

### Typisches Ergebnis

```typescript
{
  "11111111-1111-1111-1111-111111111111"
}
```

# SQL

```typescript
create or replace function public.remove_super_codex_duplicates(
  p_base_codex_ids uuid[],
  p_super_codex_ids uuid[]
)
returns uuid[]
language sql
immutable
as $function$
with base_ids as (
  select distinct unnest(coalesce(p_base_codex_ids, '{}'::uuid[])) as id
),
super_ids as (
  select distinct unnest(coalesce(p_super_codex_ids, '{}'::uuid[])) as id
)
select coalesce(
  array(
    select b.id
    from base_ids b
    left join super_ids s
      on s.id = b.id
    where s.id is null
    order by b.id
  ),
  '{}'::uuid[]
);
$function$;
```
