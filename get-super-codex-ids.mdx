## `public.get_super_codex_ids(p_base_codex_ids uuid[])`

### Zweck

Leitet aus einer Basisliste von Codex-IDs zusätzliche **Super-Codex-IDs** ab.

### Was die Funktion fachlich macht

- prüft aktive `codex_super_rules`

- betrachtet deren `trigger_entry_id`

- zieht `target_entry_id`, wenn **alle** Trigger vorhanden sind

### Aufrufparameter

| Parameter          | Typ      | Bedeutung       |
| ------------------ | -------- | --------------- |
| `p_base_codex_ids` | `uuid[]` | Basis-Codex-IDs |

### Rückgabewert

| Rückgabe | Typ      | Bedeutung                   |
| -------- | -------- | --------------------------- |
| Ergebnis | `uuid[]` | Zusätzliche Super-Codex-IDs |

### Beispiel

```typescript
select public.get_super_codex_ids(
  array[
    '11111111-1111-1111-1111-111111111111'::uuid,
    '22222222-2222-2222-2222-222222222222'::uuid
  ]
);
```

### Typisches Ergebnis

```typescript
{
  "33333333-3333-3333-3333-333333333333"
}
```

# SQL

```typescript
create or replace function public.get_super_codex_ids(
  p_base_codex_ids uuid[]
)
returns uuid[]
language sql
stable
as $function$
with input_ids as (
  select distinct unnest(coalesce(p_base_codex_ids, '{}'::uuid[])) as codex_id
),
rule_trigger_counts as (
  select
    sr.id as super_rule_id,
    sr.target_entry_id,
    count(*) as required_count
  from public.codex_super_rules sr
  join public.codex_super_rule_triggers srt
    on srt.super_rule_id = sr.id
  join public.codex_entries target_ce
    on target_ce.id = sr.target_entry_id
  where sr.is_active = true
    and sr.rule_mode = 'all_required'
    and target_ce.is_active = true
  group by sr.id, sr.target_entry_id
),
rule_match_counts as (
  select
    sr.id as super_rule_id,
    sr.target_entry_id,
    count(distinct srt.trigger_entry_id) as matched_count
  from public.codex_super_rules sr
  join public.codex_super_rule_triggers srt
    on srt.super_rule_id = sr.id
  join input_ids i
    on i.codex_id = srt.trigger_entry_id
  where sr.is_active = true
    and sr.rule_mode = 'all_required'
  group by sr.id, sr.target_entry_id
)
select coalesce(
  array(
    select distinct rtc.target_entry_id
    from rule_trigger_counts rtc
    join rule_match_counts rmc
      on rmc.super_rule_id = rtc.super_rule_id
    where rmc.matched_count = rtc.required_count
    order by rtc.target_entry_id
  ),
  '{}'::uuid[]
);
$function$;
```
