## `public.render_codex_payload(p_payload jsonb, p_root_tag text)`

### Zweck

Wandelt einen JSONB-Payload in einen XML-ähnlichen Prompt-Block um.

### Aufrufparameter

| Parameter    | Typ     | Bedeutung                                  |
| ------------ | ------- | ------------------------------------------ |
| `p_payload`  | `jsonb` | Codex-Payload                              |
| `p_root_tag` | `text`  | Root-Tag, z. B. `codex` oder `super codex` |

### Rückgabewert

| Rückgabe | Typ    | Bedeutung         |
| -------- | ------ | ----------------- |
| Ergebnis | `text` | gerenderter Block |

### Beispiel

```typescript
select public.render_codex_payload(
  '[{"codex_title":"Kalega","codex_text":"Ein Schiff","codex_details":["Alter Reaktor"]}]'::jsonb,
  'codex'
);
```

### Typisches Ergebnis

```typescript
<codex>
<codex entry name="Kalega">
Kalega
Ein Schiff
Alter Reaktor
</codex entry>
</codex>
```

# SQL

```typescript
create or replace function public.render_codex_payload(
  p_payload jsonb,
  p_root_tag text
)
returns text
language sql
immutable
as $function$
with entries as (
  select
    elem,
    ord
  from jsonb_array_elements(coalesce(p_payload, '[]'::jsonb)) with ordinality as t(elem, ord)
),
rendered as (
  select
    ord,
    array_to_string(
      array_remove(
        array[
          format('<codex entry name="%s">', coalesce(elem->>'codex_title', '')),
          coalesce(elem->>'codex_title', ''),
          coalesce(elem->>'codex_text', ''),
          case
            when jsonb_typeof(elem->'codex_details') = 'array' then (
              select string_agg(detail_text, E'\n\n' order by didx)
              from (
                select
                  didx,
                  nullif(
                    case
                      when jsonb_typeof(d) = 'string' then trim(both '"' from d::text)
                      when jsonb_typeof(d) = 'object' then coalesce(
                        d->>'codex_detail',
                        d->>'detail_text',
                        d->>'text',
                        ''
                      )
                      else ''
                    end,
                    ''
                  ) as detail_text
                from jsonb_array_elements(coalesce(elem->'codex_details', '[]'::jsonb)) with ordinality as dd(d, didx)
              ) q
              where detail_text is not null
            )
            else null
          end,
          '</codex entry>'
        ],
        null
      ),
      E'\n'
    ) as entry_text
  from entries
)
select array_to_string(
  array_remove(
    array[
      format('<%s>', p_root_tag),
      (select string_agg(entry_text, E'\n' order by ord) from rendered),
      format('</%s>', p_root_tag)
    ],
    null
  ),
  E'\n'
);
$function$;
```
