## `public.evaluate_story_variable_condition(p_variable_type text, p_left_value jsonb, p_operator text, p_right_value jsonb)`

### Zweck

Bewertet **eine einzelne Regelbedingung**.

### Unterstützte Typen

- `boolean`

- `integer`

- `int`

- `bigint`

- `numeric`

- `number`

- `decimal`

- `float`

- `text`

- `string`

- `enum`

- `json`

### Unterstützte Operatoren

- `=`

- `<>`

- `!=`

- `<`

- `<=`

- `>`

- `>=`

- `like`

- `not like`

### Wichtiger Punkt

`like` wird intern **case-insensitive** behandelt, also über `ILIKE`.

### Aufrufparameter

| Parameter         | Typ     | Bedeutung          |
| ----------------- | ------- | ------------------ |
| `p_variable_type` | `text`  | Variablentyp       |
| `p_left_value`    | `jsonb` | aktueller Wert     |
| `p_operator`      | `text`  | Vergleichsoperator |
| `p_right_value`   | `jsonb` | Vergleichswert     |

### Rückgabewert

| Rückgabe | Typ       | Bedeutung                    |
| -------- | --------- | ---------------------------- |
| Ergebnis | `boolean` | Bedingung erfüllt oder nicht |

### Beispiel

```typescript
select public.evaluate_story_variable_condition(
  'enum',
  '{"value":"Demokatrie"}'::jsonb,
  '=',
  '{"value":"Demokatrie"}'::jsonb
);
```

Ergebnis:

```typescript
true
```

### Beispiel mit `like`

```typescript
select public.evaluate_story_variable_condition(
  'text',
  '{"value":"Fritz"}'::jsonb,
  'like',
  '{"value":"%TZ"}'::jsonb
);
```

Ergebnis:

```typescript
true
```

# SQL

```typescript
create or replace function public.evaluate_story_variable_condition(
  p_variable_type text,
  p_left_value jsonb,
  p_operator text,
  p_right_value jsonb
)
returns boolean
language plpgsql
immutable
as $function$
declare
  v_type text;
  v_op text;

  l_raw text;
  r_raw text;

  l_bool boolean;
  r_bool boolean;

  l_int bigint;
  r_int bigint;

  l_num numeric;
  r_num numeric;
begin
  if p_left_value is null or p_right_value is null then
    return false;
  end if;

  v_type := lower(coalesce(p_variable_type, ''));
  v_op := lower(btrim(coalesce(p_operator, '')));

  l_raw := coalesce(p_left_value ->> 'value', p_left_value #>> '{}');
  r_raw := coalesce(p_right_value ->> 'value', p_right_value #>> '{}');

  if v_type = 'boolean' then
    l_bool := l_raw::boolean;
    r_bool := r_raw::boolean;

    case v_op
      when '='  then return l_bool = r_bool;
      when '<>' then return l_bool <> r_bool;
      when '!=' then return l_bool <> r_bool;
      else return false;
    end case;

  elsif v_type in ('integer', 'int', 'bigint') then
    l_int := l_raw::bigint;
    r_int := r_raw::bigint;

    case v_op
      when '='  then return l_int = r_int;
      when '<>' then return l_int <> r_int;
      when '!=' then return l_int <> r_int;
      when '<'  then return l_int < r_int;
      when '<=' then return l_int <= r_int;
      when '>'  then return l_int > r_int;
      when '>=' then return l_int >= r_int;
      else return false;
    end case;

  elsif v_type in ('numeric', 'number', 'decimal', 'float') then
    l_num := l_raw::numeric;
    r_num := r_raw::numeric;

    case v_op
      when '='  then return l_num = r_num;
      when '<>' then return l_num <> r_num;
      when '!=' then return l_num <> r_num;
      when '<'  then return l_num < r_num;
      when '<=' then return l_num <= r_num;
      when '>'  then return l_num > r_num;
      when '>=' then return l_num >= r_num;
      else return false;
    end case;

  elsif v_type in ('text', 'string', 'enum') then
    case v_op
      when '=' then return l_raw = r_raw;
      when '<>' then return l_raw <> r_raw;
      when '!=' then return l_raw <> r_raw;
      when '<' then return l_raw < r_raw;
      when '<=' then return l_raw <= r_raw;
      when '>' then return l_raw > r_raw;
      when '>=' then return l_raw >= r_raw;
      when 'like' then return l_raw ilike r_raw;
      when 'not like' then return l_raw not ilike r_raw;
      else return false;
    end case;

  elsif v_type = 'json' then
    case v_op
      when '='  then return p_left_value = p_right_value;
      when '<>' then return p_left_value <> p_right_value;
      when '!=' then return p_left_value <> p_right_value;
      else return false;
    end case;

  else
    return false;
  end if;
end;
$function$;
```
